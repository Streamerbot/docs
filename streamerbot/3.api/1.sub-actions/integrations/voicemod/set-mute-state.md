---
title: Set Mute State
description: Modify the current muted state in VoiceMod
version: 0.1.8
parameters:
  - name: State
    type: Select
    description: Select the new muted state to set in VoiceMod
    required: true
    options:
      - value: Enabled
        description: Enable the muted state in VoiceMod
      - value: Disabled
        description: Disable the muted state in VoiceMod
      - value: Toggle
        description: Toggle the current muted state in VoiceMod
---
