# Set Paused State
Toggle the paused state of a channel point reward.

## Parameters
### `Reward`
Select the reward.

- Type: `Selection`

### `State`
- `Paused`: Pause the channel point reward
- `Unpaused`: Unpause the channel point reward
- `Toggle`: Toggle the current status between paused and unpaused
