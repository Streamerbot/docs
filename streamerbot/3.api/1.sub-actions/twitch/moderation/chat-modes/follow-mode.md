# Follow Mode
Turn follow mode on or off

## Parameters
### `State`
Turn the follow mode on or off

- `On`: Enable follow mode
- `Off`: Disable follow mode

### `Duration`
The duration of the follow mode

- Type: `Integer`
- *Optional*
