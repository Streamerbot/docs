# Slow Mode
Turn slow mode on or off

## Parameters
### `State`
Turn slow mode on or off

- `On`: Enable slow mode
- `Off`: Disable slow mode

### `Duration`
The duration of the slow mode

- Type: `Integer`
- *Optional*
