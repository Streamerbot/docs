# Set Replay Buffer State
Change your replay buffer status
:image-preview

## Parameters
:parameter{name=ObsConnection}

### `State`
Choose the recording state

- `Start`: Start your replay buffer
- `Stop`: Stop your replay buffer
- `Save`: Save your replay buffer
