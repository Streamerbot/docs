Integrate Streamer.bot with [DonorDrive](https://donordrive.com)

## Configuration

::steps{level=3}

### Add a DonorDrive service to Streamer.bot

:::navigate
Navigate to **Integrations > DonorDrive** in Streamer.bot
:::

1. :kbd{value="Right-Click"} anywhere in the panel and select `Add`
2. Enter a name for this DonorDrive service
3. Select a charity provider, or enter a custom API endpoint
