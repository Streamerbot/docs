## Overview

If you are coming from another bot platform, you are probably used to having some basic commands available out of the box.

::note
**Streamer.bot ships with a blank slate**, but a few simple actions can provide similar functionality.
::

Below, you will find a code to import some basic chat commands directly into your Streamer.bot instance for immediate use.

## Instructions

::steps{level=3}

### Load the import code

1. Copy the import code above
1. Click **Import** at the top of Streamer.bot to open the Import Dialog
1. Paste the code into the **Import String** field

### Confirm the Import

You should then see the **Name** field populated with `Quick Start - Chatbot Example`, and numerous actions and commands to import should now show in the list.

:::warning
For security reasons, all commands are disabled by default. **You must manually enable them before they will work.**
:::

Click **Import** to finalize your selections

:::success
All selected actions will be added to your Streamer.bot installation!
:::
